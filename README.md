# Морской бой (Sea Battle) — Сетевая игра

Клиент-серверное приложение для игры в «Морской бой» по сети. Проект реализован на языке C++ с использованием фреймворка Qt.

---

## Системные требования и окружение

Для успешной компиляции и запуска проекта необходимо следующее окружение (на выбор):

### Вариант 1: Windows
- ОС: Windows 10 / 11 (64-bit)
- Компилятор: MSVC 2022 (v143) или MSVC 2019 (v142)
- Среда разработки: Qt Creator и Visual Studio 2022
- Система сборки: CMake версии 3.16 или выше

### Вариант 2: Linux
- ОС: Ubuntu 24.04 LTS или новее
- Компилятор: GCC 11+ или Clang 14+
- Среда разработки: Qt Creator и VS Code
- Система сборки: CMake версии 3.16 или выше

---

## Зависимости и дополнительные библиотеки

1. Qt 6 Framework (рекомендуемая версия: 6.8.3 LTS).
   - Обязательные модули: Qt6::Widgets, Qt6::Multimedia (для воспроизведения звуковых эффектов).
2. Системные сетевые библиотеки: 
   - Windows: ws2_32 (Winsock2).
   - Linux: POSIX Socket.

## Последовательность сборки приложения

### Подготовка репозитория
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/Bananka123321/Sea_Battle.git
   cd Sea_Battle
   ```

### Сборка под Windows

1. Установите Qt Online Installer.
2. При установке выберите версию Qt 6.8.3+, архив msvc2022_64 и обязательно поставьте галочку напротив модуля Qt Multimedia.
3. Откройте Qt Creator → Файл → Открыть файл или проект... → выберите CMakeLists.txt из папки Sea_Battle_Client.
4. В настройках набора выберите Desktop Qt 6.7.x MSVC2022 64bit.
5. Нажмите кнопку Собрать или Ctrl+B. Исполняемый файл Client.exe появится в папке build/.

### Сборка под Linux (Ubuntu)

1. Установите необходимые системные зависимости и пакеты разработки Qt:
   ```bash
   sudo apt update
   sudo apt install -y build-essential cmake \
       qt6-base-dev qt6-multimedia-dev libqt6multimedia6 \
       libgl1-mesa-dev libxkbcommon-x11-0 libxcb-cursor0 \
       libxcb-icccm4 libxcb-image0 libxcb-keysyms1 \
       libxcb-randr0 libxcb-render-util0 libxcb-shape0 \
       libxcb-xinerama0 libssl-dev
   ```

2. Перейдите в корневую папку проекта и выполните сборку:
   ```bash
   cmake -S Sea_Battle_Client -B Sea_Battle_Client/build -DCMAKE_BUILD_TYPE=Release
   cmake --build Sea_Battle_Client/build --config Release --parallel
   ```

Исполняемый файл Client будет находиться в папке Sea_Battle_Client/build/.

## Запуск приложения

- Сервер: Убедитесь, что сервер запущен и доступен.
  - Для использования публичный адрес: 31.192.108.37:7070.
  - Для локального тестирования: запустите исполняемый файл Server из папки сервера, в клиенте в файле tcp_client.cpp измените адрес сервера с 31.192.108.37 на 192.168.0.1.
- Клиент: Запустите собранный исполняемый файл Client.
  - Введите имя пользователя и подключитесь к серверу, создав новую игру или присоединившись к существующей по коду лобби.

## Структура проекта

- Sea_Battle_Client/ — Исходный код клиентской части.
  - UI/ — Файлы интерфейса.
  - App/ — Логика игровых сцен, досок и кораблей.
  - Network/ — Классы для TCP-соединения и маршрутизации сообщений.
- Server/ — Исходный код серверной части, обрабатывающий сессий подключений, лобби, игровых сессий.
- common/ — Общие заголовочные файлы для клиента и сервера, включающий протокол, валидацию, JSON.
